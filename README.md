# LoveOS: Interactive Web-Based Desktop Experience 💝

**LoveOS** is a gamified, Valentine-themed web operating system simulation. This project serves as an interactive UI/UX template and a technical showcase of complex DOM manipulation, emotional design principles, and modern frontend architecture.

![LoveOS Preview]([https://via.placeholder.com/800x450.png?text=LoveOS+Desktop+Preview](https://ank77r.github.io/Project-Affinity-Romantic-UI-UX-Template/)) ## 📌 Project Overview
The goal of this project was to step away from traditional, static web layouts and create a highly interactive, stateful environment. It simulates a desktop OS right in the browser, complete with draggable windows, interactive applications, and a gamified user journey. 

This template is perfect for developers or designers looking to showcase emotional design—using micro-interactions, animations, and narrative to create a memorable user experience.

## ✨ Key Features
* **Custom Window Management System:** Fully draggable app windows with dynamic `z-index` layering (click-to-focus) and minimize/close functionality.
* **Glassmorphism UI:** Modern, frosted-glass aesthetics with CSS backdrop-filters, custom animations, and responsive layout handling.
* **Gamification:** Integrated "Heart Catcher" mini-game utilizing collision detection, timers, and dynamic DOM element spawning.
* **Simulated Terminal:** A custom command-line interface that processes specific text inputs to trigger UI events.
* **Hardware API Integration:** Implementation of the Web Speech API for voice recognition and file inputs for simulated photo verification.
* **Stateful Animations:** Complex CSS keyframe animations, 3D CSS transforms (tilt-cards, polaroid flips), and JS-triggered confetti effects.

## 🛠️ Tech Stack
This project was built focusing on core web technologies to ensure maximum performance and zero heavy framework dependencies.
* **Markup & Styling:** HTML5, CSS3 (Variables, Flexbox, CSS Grid, Keyframes)
* **Logic & Interactivity:** Vanilla JavaScript (ES6+), jQuery & jQuery UI (for drag-and-drop physics)
* **Third-Party Libraries:** [Canvas Confetti](https://github.com/catdad/canvas-confetti) for celebration effects.
* **Icons & Fonts:** FontAwesome, Google Fonts.
